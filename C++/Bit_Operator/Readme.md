# Bit Operator

* & - bitwise AND
  * true AND true = true
  * true AND false = false
  * false AND false = false



* | - bitwise OR
  * true OR true = true
  * true OR false = true
  * false OR false = false



* ^ - bitwise XOR (Exclusive OR)
  * true OR true = false 
  * false OR false = false
  * true OR false = true



* ~ - bitwise NOT
  * 1 change to 0 and 0 change to 1 



* `>>` - bitwise SHIFT
  * In case of `>>n` or `<<n` , moved to n bits
